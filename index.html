Маркований список з різними маркерами та зображеннями:

  • Перший елемент звичайного маркованого списку
  • Другий елемент звичайного маркованого списку
  • Третій елемент звичайного маркованого списку

Нумерований список із різною нумерацією:

  1. Перший елемент нумерованого списку (тип 1)
  2. Другий елемент нумерованого списку (тип 1)
  3. Третій елемент нумерованого списку (тип 1)
  1. Перший елемент нумерованого списку (тип A)
  2. Другий елемент нумерованого списку (тип A)
  3. Третій елемент нумерованого списку (тип A)

Багаторівневий список:

  • Елемент 1
  • Елемент 2
  • Елемент 3
    • Піделемент 3.1
    • Піделемент 3.2
  • Елемент 4

Список визначень:

HTML
Мова розмітки гіпертексту, використовується для створення веб-сторінок.
CSS
Каскадні таблиці стилів, використовуються для оформлення веб-сторінок, написаних на HTML.
JavaScript
Мова програмування, використовується для надання інтерактивності на веб-сторінках.
